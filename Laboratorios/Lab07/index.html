BULMA - Frameworl

Bulma es un framework CSS sencillo, moderno, ligero y elegante basado en Flexcaja que está conformado por una colección de clases.
Este framework puede configurarse de dos modos distintos, desde el instalador npm o descargarlo directamente de la documentación enlazandolo al CDN de nuestro archivo HTML, lo cual nos permitirá acceder a las clases de Bulma.
Una de las principales caracteristicas de Bulma es la forma en que maneja las columnas en las vistas de los sitios webs. Recordando que es un framework basado en Flexcaja, permitirá que le ofrezcamos al usuario una experiencia bastante agradable en el modo responsivo del sitio web.

Por qué elegir Bulma?

  • Es fácil de aprender
  • Es modular
  • Es solamente para CSS
  • Es responsivo
  • Esta bastante bien documentado
  • Es compatible
  • Es mejorando constantemente

Material Design

Según Google, Material Design o Diseño Material, es un lenguaje que combina los principios innovadores de la tecnología con las normas clásicas del diseño. Es decir, se sintetizan los principios clásicos del buen diseño. De esta forma, como desarrolladores se nos facilita la ccreación de sitios web o aplicaciones moviles en relación al diseño, usabilidad y calidad, permitiendo que las aplicaciones se puedan adaptar a cualquier dispositivo electronico como tabletas, televisores, autos, laptops, etc.
