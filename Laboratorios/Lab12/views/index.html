Describe el archivo package.json.

El archivo Package.JSON es un archivo de definición o manifiesto, en donde se especifican las referencias al proyecto como: autor, repositorio, version, las dependencias, etc.
En simples palabras, es el archivo en formato JSON donde se colocará toda la información sobre nuestro proyecto y gracias a este se puede recuperar información útil de nuestro proyecto.
